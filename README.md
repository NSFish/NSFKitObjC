## Example

To run the example project, clone the repo, and run `pod install` from the Example directory first.

## Requirements

iOS 9.0 or above.

## Installation

Add the following line to your Podfile:

```ruby
pod 'NSFKitObjC', :git => 'git@github.com:NSFish/NSFKitObjC.git'
```
